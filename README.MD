# Лабиринт сокровищ

Текстовая приключенческая игра, созданная в рамках учебного проекта.

## Описание

Исследуйте таинственный лабиринт, решайте загадки, собирайте предметы и найдите главное сокровище!

## Установка

```bash
# Установка зависимостей
make install
# или
poetry install
# Запуск игры
make project
# или
poetry run project
```

## Управление
```
    go <direction> - перемещение (north, south, east, west)

    look - осмотр текущей комнаты

    take <item> - взять предмет

    use <item> - использовать предмет

    inventory - показать инвентарь

    solve - решить загадку

    help - справка по командам

    quit - выход из игры
```
